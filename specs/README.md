# Component Specs

This directory is the local SDKWork component contract for `sdkwork-connect`.

- Component root: `sdkwork-connect`
- Canonical standards: `../sdkwork-specs/README.md`
- Machine-readable contract: `specs/component.spec.json`

Read `specs/component.spec.json` before changing public exports, runtime entrypoints, SDK clients, generated artifacts, config keys, repository package layout, or verification commands.

Do not copy root standards into this directory. Link to files under `../sdkwork-specs/` instead.
