# SDKWork Connect Workspace

This `.sdkwork/` directory is source-controlled workspace metadata for the SDKWork Connect repository.

It may contain repository-local skills, plugins, and optional non-secret manifests. It is not runtime state and must not contain generated SDK output, local service data, peer private keys, user files, logs, caches, tokens, certificates, or secrets.

Authoritative root standards are under `../sdkwork-specs/`.
